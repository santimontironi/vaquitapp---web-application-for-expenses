# Contexto - VaquitApp

## Qué es este proyecto

La idea de este proyecto es una app web que sea de organizar los gastos de una salida o plan que incluye a varias personas. Una vez registrado y logueado podes crear grupos o abrir los grupos a los que formas parte. En cada grupo podes hacer planes por ej "asado los domingos". Para crear cada plan pones el nombre y los usuarios que forman parte del mismo, añadiendo tambien el gasto total que sale el plan. Luego la app internamente debe de hacer el calculo de cuanto debe de pagar cada uno o cuanto le tienen que dar a/os usuarios que pagaron solo ellos por los demas. 

## Estructura de carpetas estimadas

BACKEND (NODE.JS, EXPRESS.JS y MONGODB): carpetas config, controllers, middlewares, models, repository y routes. luego archivos como: .env, gitignore, app.js, index.js, server.js y los package-json

FRONTEND (REACT + TYPESCRIPT + TAILWIND CSS, creado con Vite): carpetas context, components, pages, assets y utils.

*LA IDEA ES TERMINAR EL BACKEND PRIMERO ANTES DE ARRANCAR EL FRONTEND.*
